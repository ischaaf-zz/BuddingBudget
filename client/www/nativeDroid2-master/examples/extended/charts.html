Chart Themes

Usually charts with class .nd2-chart will be themed with the primary color. You can overwrite this behavior individually for any chart-block by adding a class called .clr-theme-COLORNAME. You can set any color specified in the Color section.

.clr-theme-lime
.clr-theme-pink
.clr-theme-indigo
